## Project Idea

My initial project idea is to build a simple smart bird feeder monitoring device that can distinguish between birds, squirrels, and empty scenes using embedded image classification. The problem domain is garden wildlife monitoring, with a focus on detecting squirrel intrusion at a bird feeder without relying on cloud computing. I plan to use computer vision and transfer learning in Edge Impulse to train a lightweight TensorFlow model on labelled images of birds, squirrels, and empty feeder-related scenes. The main data will come from self-collected and curated reference images, covering different poses, lighting conditions, and backgrounds. For the physical platform, I expect to use an Arduino Nano 33 BLE Sense with an OV7675 camera and a Tiny Machine Learning Shield, so that the device can capture images locally, run inference on the board, and provide a simple physical response such as LED feedback. At this early stage, the goal is to keep the idea achievable and focus on a fixed-view classification task rather than a more complex real-time object detection system.
